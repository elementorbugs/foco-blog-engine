FOCO Features — What the App Does for ADHD Brains

FOCO has six core features, each built for a specific ADHD struggle: task breakdown for paralysis, voice capture for when typing is too much, focus mode for attention drift, brain dump for working memory overload, time estimates for time blindness, and a mascot companion for body doubling. One narrow tool, six precise applications.

The 6 FOCO features at a glance

  • Task Breakdown — one stuck task in, five tiny first steps out, with time estimates.
  • Voice Capture — speak instead of type when typing feels like too much.
  • Focus Mode — built-in timers calibrated for ADHD attention spans.
  • Brain Dump — externalize every open loop in your head in five minutes.
  • Time Estimates — calibrated per-step estimates that fight time blindness.
  • Mascot Companion — a quiet presence that stays with you while you work.

Why FOCO has six features and not sixty

Most productivity apps add features to feel competitive. FOCO has six because each one addresses a specific, documented ADHD executive function gap. We will not add a feature just because users ask for it. We will add it if it closes a wall ADHD brains hit repeatedly.

The narrow scope is the point. FOCO is the precision instrument, not the Swiss army knife.

You start with a stuck task. Voice capture or type it in. FOCO breaks it down with time estimates. You pick the first step. Focus mode runs a timer calibrated for that step. The mascot stays present. When you finish, FOCO surfaces the next step.

That's the whole loop. Six features, one workflow, no decision fatigue.

FAQ

Do I get all 6 features in the same plan?

Yes. Both Monthly and Yearly tiers include every feature. There is no premium tier that locks features. Full pricing details →

Are any features available without the app?

Yes — focus timers, brain dump, and body doubling are available free on the website at /adhd-tools/. The deeper integration (task breakdown, voice capture, time estimates per task) is only in the mobile app.

What if I only want one feature?

Honestly — if you only want the timer, use ours for free on the website and you don't need to subscribe. FOCO's value is in the workflow that connects the features, not any single one in isolation.

Are the features built for ADHD or just marketed for ADHD?

Built for. Every feature exists because the founder hit the specific wall it solves. Read the founder story for the full origin.

How is this different from Todoist or Things 3?

Those are task managers that assume you can start tasks. FOCO assumes you cannot — and addresses that gap specifically. Different problem space, different design choices.
