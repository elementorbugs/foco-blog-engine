Goblin Tools vs Tiimo: Which ADHD Task App Actually Wins?

Goblin Tools is the simpler, faster choice for ADHD task breakdown — type a task, get five tiny steps, free on the web. Tiimo is the deeper, more structured choice for ADHD time management — visual schedules, focus timer, mood tracking, but most useful features require Pro (~$6/mo). Pick Goblin Tools for unsticking yourself in the moment. Pick Tiimo if your day needs a visual map.

Key Takeaways

  • Goblin Tools is built around one job — breaking a task into smaller pieces. It does that job well and is free on the web.
  • Tiimo is built around the whole day — visual timeline, schedule, focus timer, mood. The AI task breakdown only unlocks with Pro.
  • For acute task paralysis ("I cannot start this"), Goblin Tools wins on speed and zero friction.
  • For chronic planning struggle ("I cannot structure my day"), Tiimo wins on depth and persistence.
  • Neither is built specifically for adult ADHD task initiation deficit. That is a different, narrower problem space.

What is Goblin Tools?

Goblin Tools is a free web app built around one core idea: ADHD brains get stuck because tasks feel too big, and the answer is to make them smaller.

The flagship feature is Magic ToDo. You type a task — "clean kitchen," "write report," "do laundry" — and Goblin Tools breaks it into five small steps. You can click a step and break it down further if it is still too big.

The interface is deliberately ugly. Plain checkboxes, no animations, no streaks, no gamification. The developer (who built it for their own ADHD partner) has been explicit about this — every visual polish you add is a chance for the brain to get distracted.

Beyond Magic ToDo, there are eight other tools — Formalizer (rewrites text in a different tone), Judge (emotional analysis), Taskmaster (single-task focus), Estimator (time estimation), Compiler (organize brainstorm dumps), and a few others. Most ADHD users only ever touch Magic ToDo.

The web version is free with no account required. The mobile app is a one-time ~$0.99 purchase. There is no subscription. There is no upsell. There is no usage limit you will ever realistically hit.

What is Tiimo?

Tiimo is a visual day-planner app built by and for neurodivergent users, with strong roots in executive function research.

The core feature is the visual timeline. You schedule tasks onto a day-long timeline with color-coded blocks, icons, and time estimates. The visual structure is the point — Tiimo's bet is that ADHD and autistic users plan better when they can see the shape of the day.

Pro features unlock AI task breakdown (similar in spirit to Goblin Tools but more deeply integrated with the schedule), custom widgets for iOS and Android home screens, and more advanced scheduling options. Pro costs around $6 per month with a 7-day free trial.

Tiimo also includes a focus timer, mood tracking, and streak/reward systems for finishing scheduled blocks. It is more of an all-day companion than a single-purpose tool.

Platform reach is broader than Goblin Tools — iOS, iPadOS, watchOS, Android (recently relaunched), and a desktop web app for planning sessions.

How do they break down tasks differently?

This is the dimension where ADHD adults most often choose between the two, and the difference is real.

Goblin Tools treats task breakdown as a single isolated job. You type one task, you get five steps, you close the tab. There is no persistence, no schedule, no follow-up. The whole interaction takes 15 seconds.

This is the right design when the failure mode is task initiation deficit — your brain cannot move from "knowing" to "doing" because the task feels too big. Five smaller steps lower the threshold enough to start. Goblin Tools wins for unsticking this moment.

Tiimo treats task breakdown as one step in a bigger planning system. The AI breakdown puts the subtasks onto your visual timeline with estimated time blocks. You see when each step happens. The breakdown is persistent — you come back to it tomorrow and the structure is still there.

This is the right design when the failure mode is structural — you have ten things to do, you cannot see how they fit in a day, and the chaos itself is paralyzing. Tiimo wins for organizing the whole week.

Most ADHD adults need both at different moments. Few will use both apps every day.

Which is better for adult ADHD task initiation?

For pure task initiation — the specific moment when you cannot start something — Goblin Tools wins.

Three reasons. First, zero friction. You open a web page and type. There is no login, no setup, no choosing colors, no scheduling. ADHD adults who try Tiimo for task initiation often quit during the first onboarding because the setup itself becomes a procrastination object.

The Russell Barkley executive function model describes ADHD as a problem of self-regulation. Adding more steps to the act of starting amplifies the problem.

Second, no commitment. Goblin Tools does not ask you to plan your whole day. It asks "what is the one thing stuck right now?" That bounded scope is much easier for an ADHD brain to engage with.

Third, free. You can use Goblin Tools 50 times today at no cost. Trying it has no friction either financially or emotionally. The barrier to use is literally zero.

That said — Tiimo's task breakdown, when you actually use it, produces more useful output than Goblin Tools. The subtasks come with time estimates and slot into your schedule. If you can get past the setup, the result is more actionable. The catch is whether the setup ever happens.

How do the free tiers compare?

The free tier comparison is more lopsided than most reviews acknowledge.

Goblin Tools free is the full product. Magic ToDo, all nine tools, web access, no limits. The only thing the paid mobile app adds is convenience — having the tools on your phone without opening a browser. The free web version does everything you actually need.

Tiimo free is intentionally limited. You get the basic visual planner, to-do list, and focus timer. The AI task breakdown — which is the feature most ADHD adults would actually want — is locked behind Pro. Custom widgets, advanced scheduling, and some integrations are also Pro-only.

This matters because the Tiimo free trial gives you 7 days of Pro features, then drops you back to the limited free tier. Many ADHD adults find the limited free tier frustrating after experiencing Pro, but the $6/month cost can feel like a lot for a planning app.

If budget is the deciding factor, Goblin Tools wins on free-tier value. If the breakdown quality and scheduling integration matter more than the price, Tiimo Pro is worth the trial. Verify current pricing on the Tiimo site before subscribing — pricing changes occasionally.

Which has better visual scheduling?

This is not a real comparison — Goblin Tools does not have visual scheduling at all.

If visual scheduling is what you need — color-coded time blocks, icons for tasks, a day or week view that shows the shape of your time — Tiimo is the only option here.

The visual layer matters for some ADHD brains more than others. People who struggle with working memory leaks often benefit enormously from seeing the day laid out spatially. People who struggle more with starting individual tasks often do not need or use visual scheduling at all.

Tiimo's specific implementation is well-executed. The color palette is customizable, the icon set is broad enough to feel personal, and the timeline scrolls smoothly. It does what visual scheduling promises.

If you have tried calendar apps, paper planners, and bullet journals and none of them stuck, Tiimo's specific visual approach might be the version that works. If you have never wanted a visual planner, Tiimo will not convert you.

Which is cheaper long-term?

Goblin Tools costs $0 to $0.99 ever. That is the total lifetime cost.

Tiimo costs $6 per month for Pro, which is the version that includes the features most ADHD adults want. Over a year, that is roughly $72. Over five years, $360.

This is not a fair fight on raw cost. Goblin Tools is essentially free; Tiimo is a real subscription.

The fair question is value per dollar. If you use Tiimo Pro daily and it materially improves your week, $6/month is one of the cheapest interventions you can buy for adult ADHD. If you use it twice a month, it is overpriced.

The honest middle is: try the 7-day Pro trial, see if it sticks for a week, then decide. ADHD adults who can predict their own future usage are rare, so the trial is the best signal you will get.

When should you pick Goblin Tools?

Pick Goblin Tools when any of these are true.

You are stuck on a specific task right now and need to start it in the next 10 minutes. The breakdown is fast, the interface is friction-free, and you can ignore everything except the one thing that is blocking you.

You have tried other apps and quit during setup. ADHD-friendly is often a marketing claim. Goblin Tools is genuinely ADHD-friendly in that it has almost no setup at all. If onboarding is your wall, Goblin Tools removes it.

You do not want a recurring subscription. You will use this tool intermittently — when you are stuck — and you do not want to pay monthly for the privilege.

Your problem is starting, not structuring. If your days fall apart because you cannot start the first thing, Goblin Tools is calibrated for that. It does not pretend to solve the rest.

When should you pick Tiimo?

Pick Tiimo when any of these are true.

You are a visual learner who has tried text-based to-do lists and they do not stick. The timeline format is the differentiator. Tiimo will not convert someone who hates visual planners; it will work brilliantly for someone who is wired for them.

You have a routine you want to maintain, not just an emergency to solve. Tiimo is built for daily rhythm — morning, school run, work blocks, gym, evening. If you have predictable structure that needs visual reinforcement, Tiimo earns its keep.

You can budget around $6 a month and you will actually open the app daily. If you set up Tiimo and use it twice in two weeks, you are wasting the subscription. If you set it up and check it eight times a day, $6 is trivial.

Your problem is structural, not initiation-specific. If your ADHD struggle is "I cannot organize my whole day" rather than "I cannot start this one task," Tiimo is the deeper tool.

Are there better alternatives?

Yes — neither Goblin Tools nor Tiimo was built specifically for ADHD task initiation deficit, which is the narrow problem of moving from knowing to doing on a specific task.

Goblin Tools comes close. Its Magic ToDo is essentially a task initiation tool. But it stops there. Once you have the five steps, it does not help you start the first one. It does not check in. It does not show you that you are 47 minutes in and have not opened the document. It cannot.

Tiimo is broader. Its scheduling and breakdown together cover more of the ADHD productivity surface, but the trade-off is depth — none of the features are specialized for the moment you cannot start.

FOCO is built specifically for that moment. Like Goblin Tools, FOCO breaks one stuck task into smaller steps. Unlike Goblin Tools, FOCO stays with you while you do them — time estimates per step, soft check-ins as you go, and a finish line that triggers a real completion signal. It is the narrow tool for the narrow problem.

If task initiation is your specific issue, try FOCO. If task initiation is not your specific issue, neither FOCO nor Goblin Tools will be the right pick — Tiimo or a different category of tool will fit better.

FAQ

Is Goblin Tools really completely free?

The web version is fully free with no account required. The mobile app is a one-time ~$0.99 purchase, not a subscription. There is no premium tier, no usage limit, and no advertising on the site itself. Verify current pricing on the official site before downloading.

Is Tiimo worth the $6/month?

It depends on usage frequency. If you open Tiimo daily and reschedule blocks throughout the week, $6 is excellent value. If you set it up and forget it within a month, you are paying for an unused tool. Use the 7-day free trial to predict your actual usage.

Can I use both Goblin Tools and Tiimo together?

Yes, and many ADHD adults do. Goblin Tools for breaking down stuck tasks in the moment, Tiimo for visual day planning. The tools do not directly integrate, but they solve different problems and do not conflict.

Does Goblin Tools have a focus timer?

Not really. The Taskmaster tool keeps a single task on screen, which approximates focus, but there is no countdown timer or session tracker. If a focus timer matters, Tiimo or a standalone Pomodoro app is a better choice.

Does Tiimo work for non-ADHD users?

Yes. Tiimo was built for neurodivergent users but the visual planner format works for anyone who prefers spatial scheduling over text-only to-do lists. Many users without ADHD diagnoses use it successfully.

Which app handles emotional regulation better?

Tiimo has integrated mood tracking and reflection prompts. Goblin Tools has a "Judge" tool for emotional analysis of text but no mood tracking. For ADHD adults with significant emotional regulation challenges, Tiimo's persistent tracking is more useful.

Can I export my data from either app?

Goblin Tools data is essentially ephemeral — it lives in your browser session and is not designed to be exported. Tiimo data can be exported within the app. If long-term data continuity matters, Tiimo is the safer choice.

Which app is better for ADHD students?

Tiimo, by a meaningful margin. Students benefit from visual scheduling around classes, study blocks, and assignment deadlines — exactly what Tiimo is built for. Goblin Tools helps when a specific assignment feels too big, but it does not replace a planner.

The Bottom Line

Pick Goblin Tools if your problem is starting individual tasks. Pick Tiimo if your problem is structuring your whole week.

If your problem is specifically task initiation deficit — the moment-to-moment failure to move from intention to action on a particular thing — neither is built for that. Body doubling, the FOCO app, or a different category of tool will fit better.

The best version of yourself is not the one who uses the perfect app. It is the one who uses the right tool for the right moment, and recognizes when the moment changes.
